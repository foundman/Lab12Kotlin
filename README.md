# Лабораторная работа 12 (скриншоты в 12.docx)
## Было создано приложение с пагинацией на основе ЛР 11, которое:
- Добавляет зависимости Paging 3 (paging-runtime, paging-compose).
- Реализует PagingSource<Int, Post>, который по номеру страницы и размеру порции загружает данные через Retrofit (метод getPosts(page, limit)).
- Настраивает Pager с конфигурацией (размер страницы 20) и предоставляет Flow<PagingData<Post>>.
- В Jetpack Compose собирает поток через collectAsLazyPagingItems() и отображает список в LazyColumn с автоматической подгрузкой при прокрутке.
- Обрабатывает состояния загрузки (refresh, append): показывает индикаторы при дозагрузке, а при ошибке — сообщение и кнопку повторной попытки (posts.retry()).
